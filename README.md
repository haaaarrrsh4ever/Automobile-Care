# 🚗 Automobile Care - Service Centre Management System

## 📌 Description
Automobile Care is a web-based application designed to manage vehicle service operations such as customer records, service booking, job tracking, and billing. It helps automate and organize daily activities of a service center.

---

## 🎯 Features
- Customer Registration & Management
- Vehicle Service Booking
- Service Status Tracking
- Job Card Management
- Billing System
- Admin Dashboard

---

## 🛠️ Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL
